# dotfiles

Repository to store system configuration files, for backup and
reusability.

## Shell configuration

* .profile
* .zshrc
* .zsh_history

## Vim configuration

* .vim/

```bash
.vim/
├── colors
│   └── gruvbox.vim
├── pack
│   ├── flake8
│   │   └── start
│   │       └── vim-flake8 -> /home/sam/bin/vim-flake8/
│   └── gruvbox
│       └── opt
│           └── gruvbox -> /home/sam/bin/gruvbox
├── sh
│   ├── note_count.sh
│   ├── note_gen.sh
│   ├── time.sh
│   └── today.sh
├── snippets
│   ├── hashline
│   ├── skeleton.html
│   └── skeleton.md
├── swap
├── viminfo
├── viminfo.tmp
├── viminfz.tmp
└── vimrc
```
